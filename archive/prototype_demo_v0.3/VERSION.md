- **上一版本**：demo v0.2（`../prototype_demo_v0.2/`）

## 本版本范围（v0.3）
在 v0.2「流程视图外壳 + 六环节闭环链路」基础上，对三个排序/估算类 Agent 的**第 4 步精排环节做交互升级**：由「只能采用系统推荐 Top-1」改为「**所有策略下、所有候选均可点击选用并下传给下游**」，让运营可在 AI 排序结果之上做人工干预与改选。

**选品 → 选医生 → 目标人群 → 话术 → 短视频制作 → 短视频分发**

### 页面清单
| 文件 | Agent | 说明 |
|------|-------|------|
| `app.html` | 流程视图外壳（主入口） | 顶部按流程顺序排列 6 个 Agent + 下方 iframe 承载功能页；切换/高亮同步/完成态 ✓ 徽标 |
| `index.html` | 工作台 / 只读汇总 | 侧栏导航（置顶「流程视图」入口）+ 发起任务确认页 |
| `product.html` | 选品 Agent | 意图构建 → 召回 → 向量匹配 → 多因子打分 → **策略精排（候选全可选用）** → 下传选医生 |
| `doctor.html` | 选医生 Agent | 检索构建 → 召回 → 向量打分 → **LLM 精排（候选全可选用）** → 策略实时重排 → 下传人群 |
| `audience.html` | 目标人群 Agent | 信号融合 → 画像 → 平台标签 → **人群包估算（档位全可选用）** → 联动下游话术 Agent |
| `script.html` | 话术 Agent | 上游融合 → RAG → Prompt → 生成 → 合规审查（不通过自动打回步骤①重跑）→ 联动短视频 |
| `video.html` | 短视频制作 Agent | 接收话术 → 分镜拆解 → TTS配音 → 数字人合成 → 成片多模态合规复审 → 写入成片信号、移交分发 |
| `distribute.html` | 短视频分发 Agent | 接收合规成片 → 多平台规格适配 → 人群定向+发布排期 → 多平台发布执行 → 数据回流与自优化闭环 |

### v0.3 关键特性（相对 v0.2 的变化）
- **选品 Agent · 策略精排（`product.html`）**
  - 排名列表全部候选品**可点击选用**，不再固定只用系统 Top-1
  - 选中项高亮（描边 + `✓ 已选用` 角标），未选中项显示 `点击选用`
  - 结果卡、联动选医生候选池、`localStorage` 下传均跟随**手动选用项**实时刷新
  - 结果卡标题动态提示：选中 Top-1 → 「🏆 当前策略推荐 Top-1（已选用）」；选其它 → 「✓ 已手动选用 · 当前策略第 N 名」
  - 切换策略 / 重新选品复位为该策略推荐 Top-1，之后仍可任意改选；重进恢复上次选用项
- **选医生 Agent · LLM 精排（`doctor.html`）**
  - 排名列表全部候选医生**可点击选用**，选中项高亮 + `✓ 已选用`
  - 第 5 步选定医生卡、下传目标人群 Agent 的结果跟随手动选用项实时刷新
  - 切换策略复位推荐项、重进恢复上次选用医生
- **目标人群 Agent · 人群包估算（`audience.html`）**
  - 3 个人群包档位（精准 / 均衡 / 广覆盖）**强化为显式可选**：选中档位显示 `✓ 已选用 / 已选用此人群包`，其余显示 `点击选用`
  - 选定档位的人群包实时下传话术 Agent
- 三页第 4 步标题均补充「点击任意候选即可选用」的引导文案

### 沿用 v0.2 / v0.1 的核心能力
- `app.html` 流程视图外壳：顶部流程导航条 + 下方 iframe 功能区，切换/高亮同步/完成态 ✓ 徽标
- 统一 `.stage` 纵向步骤式 UI + 顶栏运行按钮 + 上游门禁机制
- 各 Agent 通过 `localStorage` 传递选定结果，下游页面具备门禁（上游未就绪则置灰）
- 话术 Agent 合规审查打回重跑闭环（违规 → 回步骤① → 重跑 → 过审）
- 短视频成片多模态合规复审（字幕 OCR + 口播 ASR + 画面 + 口径一致性）
- 短视频分发 Agent 多平台投放与数据回流闭环
