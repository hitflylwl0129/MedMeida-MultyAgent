- **当前开发版本目录**：`../../prototype/`

## 本版本范围（v0.1）
完整打通六环节下游联动链路：
**选品 → 选医生 → 目标人群 → 话术 → 短视频制作 → 分发**

各 Agent 通过 `localStorage` 传递选定结果，下游页面具备门禁（上游未就绪则置灰）。

### 页面清单
| 文件 | Agent | 说明 |
|------|-------|------|
| `index.html` | 工作台 / 只读汇总 | 侧栏导航 + 发起任务确认页 |
| `product.html` | 选品 Agent | 意图构建 → 召回 → 向量匹配 → 多因子打分 → 精排 → 下传选医生 |
| `doctor.html` | 选医生 Agent | 检索构建 → 召回 → 向量打分 → LLM 精排 → 策略实时重排 → 下传人群（含第4步手动切策略联动第5步修复） |
| `audience.html` | 目标人群 Agent | 信号融合 → 画像 → 平台标签 → 人群包估算 → 第5步联动下游话术 Agent |
| `script.html` | 话术 Agent | 上游融合 → RAG → Prompt → 生成 → 合规审查（不通过自动打回步骤①重跑）→ 联动短视频 |
| `video.html` | 短视频制作 Agent | 接收话术 → 分镜拆解 → TTS配音 → 数字人合成 → 成片多模态合规复审 → 移交分发 |

### v0.1 关键特性
- 统一 `.stage` 纵向步骤式 UI（6 个 Agent 一致）
- 顶栏运行按钮 + 上游门禁机制
- 话术 Agent 合规审查打回重跑闭环（违规 → 回步骤① → 重跑 → 过审）
- 短视频成片多模态合规复审（字幕 OCR + 口播 ASR + 画面 + 口径一致性）
- 提示框文字阅读顺序修复（`.banner` 恢复正常文本流）
