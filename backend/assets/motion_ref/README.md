# 参考动作视频（Kling motion_control 用）

本目录用于存放"参考动作视频"——AI 会逐帧提取里面的关节/姿态，把动作迁移到医生形象图上。

## 默认文件名
`ref.mp4`（可在 backend/.env 用 `MOTION_REF_FILENAME=xxx.mp4` 覆盖）

## 视频要求
- **格式**：mp4（h264 兼容性最好），≤10 MB
- **比例**：建议 9:16 竖版（与最终成片一致），其他比例也接受但可能被裁切
- **时长**：5-10 秒最佳；过短模型动作迁移信息不够，过长浪费配额
- **画面**：
  - 一个清晰可见的人物（正面/上半身最稳）
  - 动作中性自然：站立/轻微点头/讲话/微笑等
  - 背景干净，避免多人物造成关节识别混乱

## 不需要是医生
motion_control 只迁移**动作骨架**，参考人物不需要是医生本人——同事/朋友自拍均可，
医生形象图本身从你手上的素材库里挑。

## 怎么获取
- mac 摄像头：Photo Booth 或 QuickTime → 新建影片录制 → 录 5-10 秒 → 导出
- 手机自拍：传到电脑后丢这里
- 抖音/视频号现成素材：注意版权与肖像权

## 缓存
首次使用时本地直传到腾讯云 VOD 拿 FileId，存到 `backend/.cache/motion_ref_fileids.json`。
文件 mtime/size 不变就秒回缓存；改了内容会自动重传。
