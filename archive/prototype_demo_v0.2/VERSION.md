- **归档时间**：2026-06-02 19:14
- **类型**：纯前端静态 HTML 原型快照（只读归档，请勿在本目录直接修改）

- **上一版本**：demo v0.1（`../prototype_demo_v0.1/`）

## 本版本范围（v0.2）
在 v0.1 六环节联动链路基础上，完成**整体前端结构性重构**：由「6 个独立页面 + 跳转」改造为「**顶部流程导航条 + 下方功能区随流程切换**」的统一外壳形态，并补齐分发环节使链路完整闭环：

**选品 → 选医生 → 目标人群 → 话术 → 短视频制作 → 短视频分发**

### 页面清单
| 文件 | Agent | 说明 |
|------|-------|------|
| `app.html` | **流程视图外壳（v0.2 新增·主入口）** | 顶部按流程顺序排列 6 个 Agent（带箭头连接）+ 下方 iframe 承载功能页；点击切换、子页跳转高亮同步、完成态 ✓ 徽标 |
| `index.html` | 工作台 / 只读汇总 | 侧栏导航（置顶「流程视图」入口）+ 发起任务确认页 |
| `product.html` | 选品 Agent | 意图构建 → 召回 → 向量匹配 → 多因子打分 → 精排 → 下传选医生 |
| `doctor.html` | 选医生 Agent | 检索构建 → 召回 → 向量打分 → LLM 精排 → 策略实时重排 → 下传人群 |
| `audience.html` | 目标人群 Agent | 信号融合 → 画像 → 平台标签 → 人群包估算 → 联动下游话术 Agent |
| `script.html` | 话术 Agent | 上游融合 → RAG → Prompt → 生成 → 合规审查（不通过自动打回步骤①重跑）→ 联动短视频 |
| `video.html` | 短视频制作 Agent | 接收话术 → 分镜拆解 → TTS配音 → 数字人合成 → 成片多模态合规复审 → 写入成片信号、移交分发 |
| `distribute.html` | **短视频分发 Agent（v0.2 新增）** | 接收合规成片 → 多平台规格适配 → 人群定向+发布排期 → 多平台发布执行 → 数据回流与自优化闭环 |

### v0.2 关键特性（相对 v0.1 的变化）
- **新增 `app.html` 流程视图外壳**：顶部流程导航条 + 下方 iframe 功能区，随流程切换功能界面
  - 点击顶部任一 Agent → 下方切换对应界面、该步高亮
  - 子页内「进入下游 Agent」跳转时顶部高亮自动同步（监听 iframe load）
  - 注入样式隐藏子页「返回工作台」按钮，避免 iframe 内破坏外壳
  - 已产出结果环节自动显示绿色 ✓ 徽标（读 `localStorage` 的 `sv_selected_*`）
- **新增 `distribute.html` 短视频分发 Agent**：补齐链路终点，读取上游成片 + 人群包做多平台投放与数据回流
- **改造 `video.html`**：完成时写入 `sv_selected_video`，末步按钮联动短视频分发 Agent
- **`index.html`**：侧栏置顶「🧭 流程视图」入口，并补「📤 短视频分发 Agent」链接

### 沿用 v0.1 的核心能力
- 统一 `.stage` 纵向步骤式 UI + 顶栏运行按钮 + 上游门禁机制
- 各 Agent 通过 `localStorage` 传递选定结果，下游页面具备门禁（上游未就绪则置灰）
- 话术 Agent 合规审查打回重跑闭环（违规 → 回步骤① → 重跑 → 过审）
- 短视频成片多模态合规复审（字幕 OCR + 口播 ASR + 画面 + 口径一致性）
